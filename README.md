# Time Since Last Save
Displays the the amount of time that has passed since you have last saved the game when you go to the exit menu.

<img width="2378" height="1262" alt="image" src="https://github.com/user-attachments/assets/883d7d68-9506-4ef1-a302-73c76e155546" />

Made for the latest version of The Long Dark 2.51
# Installation
Refer to this site to install MelonLoader
https://tldmods.net/install.html

After that is installed, drag TimeSinceLastSaveTLD.dll into the Mods folder located in the main directory of The Long Dark
